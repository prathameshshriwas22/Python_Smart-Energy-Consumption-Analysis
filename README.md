# Smart Energy Consumption Analysis

This project analyzes electrical energy consumption data using Python.
It helps monitor power usage and detect abnormal voltage and current conditions in electrical systems.

---

## Project Structure

Smart-Energy-Analysis
│
├── energy_analysis.py
├── energy_data.csv
└── README.md

---

## Features

* Power consumption analysis
* Voltage anomaly detection
* Current overload detection
* Graph visualization using Python

---

## Technologies Used

* Python
* Pandas
* Matplotlib

---

## Applications

* Industrial energy monitoring
* Smart grid analysis
* Electrical system monitoring

---

## How to Run the Project

1. Install Python
2. Install required libraries:

pip install pandas
pip install matplotlib

3. Run the program:

python energy_analysis.py
